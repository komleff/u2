### Project Structure

```plaintext
u2/
├── src/
│   ├── shared/          # Shared library (math, physics, protocols)
│   │   ├── U2.Shared.csproj
│   │   ├── SharedLibrary.cs
│   │   └── Tests/
│   │       └── BasicTests.cs
│   ├── server/          # Headless .NET 8 server
│   │   ├── U2.Server.csproj
│   │   └── Program.cs
│   └── client/          # Unity client (to be added in future milestones)
├── U2.sln               # Visual Studio solution
├── .editorconfig        # Code style configuration
└── .github/
    └── workflows/
        └── dotnet-build.yml  # CI/CD configuration
```

### Building the Project

**Prerequisites:**

- .NET 8 SDK

**Commands:**

```powershell
# Restore dependencies
dotnet restore U2.sln

# Build
dotnet build U2.sln --configuration Release

# Run tests
dotnet test U2.sln --configuration Release
```
