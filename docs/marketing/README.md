## Работа с конвертированными документами

Конвертированные файлы используются как **промежуточный источник** для создания канонических документов. При работе:
1. Используйте канонические документы из соответствующих разделов (pve/, specs/, gdd/)
2. Если информация из конвертированного документа актуализируется, обновляйте канонический файл
3. Конвертированные файлы служат для сверки и миграции информации
