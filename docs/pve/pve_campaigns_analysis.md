# PvE — сравнительный анализ кампаний
Статус: актуален (черновик выжимки)
Версия: 0.2
Источник: [../_converted/pve.md](../_converted/pve.md)

## Обзор
Сводный анализ структур PvE‑кампаний: типы активностей, длительность сессий, прогресс, награды и контроль сложности. Цель — выделить воспроизводимые паттерны и сформировать требования к U2.

## Термины и обозначения
- Сессия — один заход игрока (30–90 минут).
- Нить кампании — последовательность миссий/актов, объединённых сюжетом или механикой.
- Петля (loop) — повторяемый цикл «взять задачу → выполнить → получить награду → апгрейд».

## Ключевые наблюдения (конспект источника)
- Диапазон длительности: короткие миссии 10–20 мин; длинные — 45–90 мин; кампании состоят из 3–5 актов.
- Типовой набор активностей: исследование, сбор/логистика, охота/бой, оборона, эвакуация, эскорт.
- Управление сложностью: капы по времени/ресурсам, усложняющиеся модификаторы, динамическая подмешка врагов.
- Награды: кредиты/ресурсы, чертежи, косметика, репутация; крупные вехи — корабли/модули/перки.
- Критерии качества: стабильные 60 FPS, предсказуемая длительность, низкая фрустрация от рандома, ясный HUD.

## Требования к U2 (переносимые)
- Каталог миссий по ролям: бой/разведка/логистика/эскорт/охота/оборона.
- Три уровня длины: 15±5 мин (быстрые), 35±10 мин (стандарт), 70±20 мин (эпизод).
- Баланс наград: мгновенные (кредиты), накопительные (ресурсы/репутация), редкие дропы по актам.
- Модификаторы сложности: таймеры, ограничение топлива/боезапаса, «умные» паттерны AI.
- Прозрачная телеметрия: цели, прогресс, награды в HUD/журнале.

## Алгоритмы / Модель
- Шаблон миссии: JSON‑схема (тип, цели, условия провала/успеха, награды, модификаторы, сложность).
- Генератор кампании: сборка из 3–5 миссий с нарастающей сложностью и логистическими переходами.
- Контроль длительности: количество целей/волн/прыжков; SLA по времени на акт.

## Данные и конфигурация
- Каталог типов миссий и параметров (слоты наград, множители сложности, допустимые сектора/биомы).
- Таблицы наград по уровням сложности и длинам сессий.
- Матрица модификаторов (эффекты, вес, совместимость).

## Тесты и верификация
- Длительность: медиана/квантили по типам; авто‑флаги выхода за SLA.
- Удержание: доля завершений миссий/кампаний; точки схода.
- Награды: прибыль/час по активностям и сложностям.

## UX / HUD
- Единая панель целей/прогресса, карта, подсказки по ресурсам/ремонту.
- Экран итогов: время, эффективность, награды, апгрейды.
