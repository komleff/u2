## Использование архива

Архивные документы **не редактируются**. При необходимости актуализации информации:

1. Создайте новый документ в соответствующем разделе ([specs/](../specs/), [gdd/](../gdd/), [pve/](../pve/))
2. Следуйте стандартам из [STYLE.md](../STYLE.md)
3. Укажите в новом документе ссылку на исходный архивный файл в разделе "Источник"
