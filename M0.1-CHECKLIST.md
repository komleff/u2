## Prerequisites for Running

To build and test this project, you need:

1. **Required:**
   - [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

2. **Recommended:**
   - Visual Studio 2022 or VS Code with C# extension
   - Git

## Build Commands (when .NET SDK is installed)

```powershell
# Restore dependencies
dotnet restore U2.sln

# Build solution
dotnet build U2.sln --configuration Debug

# Run tests
dotnet test U2.sln --configuration Debug --verbosity normal

# Run server (placeholder)
dotnet run --project src/server/U2.Server.csproj
```
